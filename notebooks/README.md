This folder contains the chapter and appendix notebooks for *Python and AI for Asset Management*. Each notebook mirrors the structure of a chapter and focuses on a coherent set of ideas, moving from classical portfolio theory to modern machine learning and reinforcement learning applications.

## Chapter Notebooks

- `ch01_asset_management_basics.ipynb` — asset management landscape, basic portfolio objects, and data structures.
- `ch02_math_stat_preliminaries.ipynb` — mathematical and statistical preliminaries, including covariance and correlation diagnostics.
- `ch03_python_infrastructure.ipynb` — environment setup, project layout, and first backtest skeletons.
- `ch04_mean_variance.ipynb` — mean–variance portfolios, efficient frontiers, and constraint variations.
- `ch05_capm_factor_models.ipynb` — Capital Asset Pricing Model (CAPM), factor regressions, and empirical illustrations.
- `ch06_black_litterman.ipynb` — Black–Litterman framework with investor views and posterior returns.
- `ch07_data_engineering.ipynb` — feature engineering for end‑of‑day data and basic data quality checks.
- `ch08_performance_backtesting.ipynb` — performance metrics, backtesting workflow, and equity curve plots.
- `ch09_ml_workflow.ipynb` — machine learning workflow, including time‑based splits and cross‑validation.
- `ch10_linear_glm.ipynb` — linear models and generalised linear models for cross‑sectional and panel data.
- `ch11_trees_ensembles.ipynb` — tree‑based models, ensembles, and feature importance diagnostics.
- `ch12_deep_learning.ipynb` — deep cross‑sectional models and regularisation techniques.
- `ch13_sequence_models.ipynb` — sequence models for time‑series prediction and signal smoothing.
- `ch14_rl_foundations.ipynb` — reinforcement learning foundations and Markov decision process examples.
- `ch15_rl_algorithms.ipynb` — tabular and approximate reinforcement learning algorithms applied to toy allocation problems.
- `ch16_unsupervised_representation.ipynb` — clustering, dimensionality reduction, and representation learning.
- `ch17_model_risk_explainability.ipynb` — model risk diagnostics and explainability tools.
- `ch18_tech_stack_deployment.ipynb` — infrastructure, deployment patterns, and monitoring.
- `ch19_llms_assistants.ipynb` — LLMs as research and coding assistants.
- `ch20_llms_agents_value_chain.ipynb` — agents along the asset management value chain.

## Appendix Notebooks

- `appx_b_numpy_pandas.ipynb` — NumPy and pandas refresher for quant finance.
- `appx_c_sklearn_cheatsheet.ipynb` — scikit‑learn cheat‑sheet with small, focused examples.
- `appx_d_pytorch_finance.ipynb` — PyTorch refresher with finance‑flavoured examples.
- `appx_f_practical_tools.ipynb` — practical tools for project structure, profiling, and testing.
- `appx_g_repo_colab.ipynb` — working with the companion repository and Colab notebooks.

To get started, open a notebook in your preferred Jupyter environment or in Google Colab, run the cells from top to bottom, and compare the results with the corresponding chapters and slide decks.
